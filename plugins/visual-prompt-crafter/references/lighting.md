# Lighting and Illumination

> Load when user needs lighting setup descriptions, mood, or atmosphere guidance.

## Key Lighting Styles

| Style | Description |
|-------|-------------|
| Cinematic lighting | Dramatic, purposeful light and shadow |
| Rembrandt lighting | 45-degree angle, triangle on cheek |
| Rim lighting | Edge light separating subject from background |
| Backlighting | Light behind subject |
| Volumetric lighting | Rays of light through atmosphere |
| God rays | Visible light beams |
| Chiaroscuro | Strong contrast between light and dark |
| Softbox lighting | Soft, diffused light |
| Beauty dish | Soft frontal lighting |
| Ring light | Circular catchlight in eyes |

## Natural Light

| Type | Description |
|------|-------------|
| Golden hour | Warm, soft light at sunrise/sunset |
| Blue hour | Cool, soft light before sunrise/after sunset |
| Magic hour | Ideal shooting time, warm tones |
| Natural sunlight | Direct sun, harsh or soft |
| Overcast light | Diffused, soft shadows |
| Window light | Interior natural light |
| Candlelight | Warm, intimate glow |
| Firelight | Warm, flickering |
| Moonlight | Cool, soft |
| Streetlamp glow | Warm amber pools |

## Lighting Qualities

| Quality | Description |
|---------|-------------|
| Harsh shadows | Strong contrast |
| Soft shadows | Diffused, gentle |
| Diffused lighting | Even, no harsh shadows |
| High key | Bright, minimal shadows |
| Low key | Dark, dramatic shadows |
| Moody lighting | Atmospheric, emotional |
| Ethereal glow | Soft, dreamlike |
| Silhouette | Backlit, dark subject |

## Color Temperature

- Neutral (white, gray)

- Magenta and cyan (cyberpunk)
- Neon colors (bright, saturated)
