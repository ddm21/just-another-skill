# Compositions and Camera Angles

> Load when user needs camera angles, shot types, or framing guidance.

## Shot Types (Distance)

| Type | Description |
|------|-------------|
| Extreme close up | Tightest framing on face/feature |
| Close up | Head and shoulders, face fills frame |
| Medium close up | Chest up, shows expression |
| Medium portrait | Waist up, standard portrait |
| Medium shot | Knee up, shows full upper body |
| Full body shot | Entire subject in frame |
| Wide shot | Subject small in environment |
| Extreme wide shot | Epic scale, subject very small |
| Long shot | Full figure, environmental context |

## Camera Angles

| Angle | Effect |
|-------|--------|
| Eye level | Neutral, natural perspective |
| High angle | Subject looks vulnerable, smaller |
| Low angle | Subject looks powerful, imposing |
| Overhead shot | Bird's eye, complete overview |
| Dutch angle | Tilted, creates unease/disorientation |
| Hip level | Informal, casual perspective |
| Knee level | Slightly elevated, intimate |
| Worm's eye view | Extreme low angle, dramatic |

## Shot Types (Purpose)

| Type | Description |
|------|-------------|
| Establishing shot | Wide shot to set location |
| Insert shot | Close-up of specific detail |
| Two shot | Two subjects in frame |
| Dirty single | Subject with some background |
| Clean single | Subject alone, minimal background |
| Insert | Detail shot within scene |
| POV | Point of view from subject |

## Movement

| Type | Description |
|------|-------------|
| Tracking shot | Camera follows subject |
| Panning shot | Horizontal camera movement |
| Crane shot | Vertical camera movement |
| Aerial shot | From aircraft/drone |
| Drone shot | Drone perspective |
| Steadicam | Smooth handheld movement |

- Leading lines composition

- Diagonal composition
- Symmetrical placement
- Asymmetrical placement

## Character Orientation

| Orientation | Description |
|-------------|-------------|
| Looking at viewer | Direct eye contact, breaking fourth wall |
| Profile view | Side facing camera |
| Three quarter view | Partial facing |
| Back to camera | Subject faces away |
| Facing camera | Front facing |
| Looking over shoulder | Dynamic |
| Looking up/down | Vertical gaze |
| Side eye | Quick glance |
| Eyes closed | Peaceful/meditative |
