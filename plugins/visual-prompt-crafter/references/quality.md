# Quality and Realism Specs

> Load when user wants photorealistic output, rendering quality, or realism specifications.

## DO NOT Use These Keywords

These actively degrade output:
- "8K" / "4K" (use imageSize parameter instead)
- "masterpiece"
- "ultra-realistic"
- "high resolution"
- "photorealistic" (in prompts - use image generation parameters)

## Instead Use

| Instead of... | Use... |
|---------------|--------|
| "8K" | Name real cameras |
| "ultra-realistic" | "true to life" or context anchor |
| "masterpiece" | "award winning" |

## Prestigious Context Anchors

Use these to convey quality:

- "National Geographic cover"
- "Vanity Fair editorial"
- "Academy Award winning cinematography"
- "Hasselblad medium format"
- "Leica rangefinder quality"
